EqualAlign 设置box-sizing属性

宽度和高度属性只包含内容，而不包含是padding、border或margin

本项目使用CSS属性:
box-sizing: border-box;

EqualAlign 选项

threshold

阈值可以在0和1之间。当设置为1时，所有相邻项目的高度都相同。当设置为0.5时，如果大于相邻元素的一半，则对大小进行调整。 当设置为0时，不会发生任何事情。 计算的宽度和高度属性，仅包括content，但不包括padding，border或margin。 此实例设置为0.2

targetClass

此元素类将会被设置style高度样式

beforeRun

元素被设置尺寸之前执行的方法

afterRun

元素被设置尺寸之后执行的方法

var options = {   targetClass:'item',   threshold:0.2,   beforeRun:function(){},   afterRun(){}
}

如何使用:

在你也面需要的地方引入EqualAlign.js

<script src="EqualAlign.js"></script>

初始化 AlighJS.

var container = document.querySelector('.container');
var equalAlign = new EqualAlign(container,{
    targetClass: 'item',
    threshold: 1
});

选项定制

// 目标类
targetClass:'item',

// 阈值可以在0和1之间。当设置为1时，所有相邻项目的高度都相同。当设置为0.5时，如果大于相邻项目的一半，则只对大小进行调整。 当设置为0时，不会发生任何事情。
// 计算的宽度和高度，仅包括content，但不包括padding，border或margin。
threshold: 1,

// 在元素被设置尺寸之前执行
beforeRun:function(){},

// 在元素被设置尺寸之后执行
afterRun:function(){}

API methods.

// 更新EqualAlign的实例
equalAlign.update(); 

// 删除AlignJS的一个实例
equalAlign.destroy();
